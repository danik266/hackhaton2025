<h1>Установка зависимостей</h1>
<h2>в папке backend</h2>

<h2>pip install uvicorn[standard]</h2>

<h2>pip install fastapi uvicorn[standard]</h2>
<h2>при этом должен быть включен venv</h2>
<h2>После установки зависимостей, создаете новый cmd терминал и пишите<br>
uvicorn backend.main:app --reload </h2>

<h1>С перекидыванием фронта думаю можно разобраться(просто перекинуть фронт в папку с проектом)</h1>
<h2>Для frontend части, так же установка зависимостей</h2>
<h2>npm install</h2>
<h2>Далее npm run dev</h2>
